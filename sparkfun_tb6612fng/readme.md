# SparkFun TB6612FNG 双路电机驱动

H桥双路电机驱动芯片 Blockly 库，适用于小型直流电机控制。

## Description

TB6612FNG 是 Toshiba 出品的双路 H 桥电机驱动 IC，支持 VM 4.5–13.5V，最大电流 1.2A/路。

## Quick Start

1. 使用"初始化电机"积木设置 IN1/IN2/PWM/STBY 引脚
2. 使用"电机 速度"积木控制速度（-255~255）
3. 使用"刹车"积木快速停止
