非接触式 XZ 位置和手势识别传感器 Blockly 库。

## Description

ZX 传感器通过红外发射/接收检测手在传感器上方的 X 位置和 Z 高度，并识别左/右滑动等手势。

## Quick Start

1. 通过 I2C 连接传感器
2. 使用"初始化 ZX"积木
3. 在循环中检查"位置数据就绪"，然后读取 X/Z 值
