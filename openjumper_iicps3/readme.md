# PS3无线接收器

适配openjumper PS3无线接收器（编号ojmoBhp4039 ），通过该库解析IICPS3手柄接收模块的数据，支持获取手柄的按键状态、摇杆位置。

## Description

适配openjumper PS3无线接收器（编号ojmoBhp4039 ），通过该库解析IICPS3手柄接收模块的数据，支持获取手柄的按键状态、摇杆位置。

This library provides 4 Blockly blocks.
