# ESP8266 SHA1 Hash

Calculate SHA1 with the ESP8266 built-in Hash library.

## Library Info
- **Name**: @aily-project/lib-esp8266-hash
- **Version**: 0.0.1
- **Author**: ESP8266 Arduino Core Team
- **Source**: ESP8266 Arduino Core 3.1.2
