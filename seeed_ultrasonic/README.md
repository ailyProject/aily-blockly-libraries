# Ultrasonic 超声波测距传感器库

## 简介

这是一个用于SeeedStudio Grove超声波测距传感器的Blockly库。该库支持测量距离，可返回厘米、毫米或英寸单位的数值。

## 支持的硬件

- Grove - Ultrasonic Ranger (HC-SR04兼容)
- 工作电压: 3.3V / 5V
- 测量范围: 3cm - 400cm
- 分辨率: 1cm

## 功能特性

- 初始化超声波传感器（指定引脚）
- 测量距离（厘米）
- 测量距离（毫米）
- 测量距离（英寸）

## 使用示例

### 基础测距

```blocks
设置超声波传感器 超声1 引脚 D2
重复执行
    串口 串口 输出 超声波 超声1 测量距离(cm) 并换行
    延时 1000 毫秒
结束
```

## 积木说明

### 初始化块
**设置超声波传感器 [变量名] 引脚 [引脚]**
- 创建一个新的超声波传感器对象
- 参数：变量名（默认"超声"）、数字引脚

### 测量块
**超声波 [变量名] 测量距离([单位])**
- 返回距离数值
- 单位可选：厘米(cm)、毫米(mm)、英寸(inch)
- 返回值类型：数字

## 技术参数

- 默认超时时间: 1秒 (1000000微秒)
- 测量原理: 发送超声波脉冲，计算回声时间
- 计算公式:
  - 厘米: 持续时间 / 29 / 2
  - 毫米: 持续时间 * 5 / 29
  - 英寸: 持续时间 / 74 / 2
