## Supported Boards

Arduino UNO, ESP32（3.3V）

## Description

SHTC3 是 Sensirion 出品的小型低功耗 I2C 温湿度传感器，温度精度 ±0.2°C，湿度精度 ±2% RH。

## Quick Start

1. 通过 Qwiic/I2C 连接传感器
2. 使用"初始化 SHTC3"积木
3. 在循环中调用"更新测量"后读取温湿度
