## Description

RV1805 是 Micro Crystal 出品的超低功耗实时时钟，I2C 接口，内置备用电池供电，精度 ±3ppm。

## Quick Start

1. 通过 Qwiic/I2C 连接
2. 使用"初始化 RV1805"积木
3. 使用"设置时间"设置初始时间
4. 循环中调用"更新时间"后读取各时间字段
