## 支持的开发板

Arduino UNO, ESP32, ESP8266, Arduino Nano RP2040, Arduino UNO R4 WiFi 等主流开发板

## 描述

PIR（被动红外）运动传感器用于检测人体运动。当有人在检测范围内移动时，传感器输出HIGH信号。检测距离可调（默认3米，最大6米），保持时间可调（1-25秒）。

## 快速开始

1. 将PIR传感器连接到数字引脚（如D2）
2. 使用初始化块设置引脚
3. 使用读取块检测运动状态
