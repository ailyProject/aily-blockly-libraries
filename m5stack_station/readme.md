# Station Onboard Ports

Board-specific control for Station's five Grove power groups, USB-A output and six INA3221 monitor channels.

The official channel mapping is fixed as INA0 CH1/CH2/CH3 = A1/A2/B1 and INA1 CH1/CH2/CH3 = B2/C1/C2. A1 and A2 share one power switch. USB-A power is controlled by GPIO12.
