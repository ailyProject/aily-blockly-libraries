# rc-switch

Operate 433/315 MHz remote controlled devices from Arduino-compatible boards.

## Supported Boards

AVR, ESP8266, ESP32, STM32, and compatible Arduino cores.

## Description

This wrapper covers transmit, receive, protocol setup, pulse length, repeat count, and received packet fields.

## Quick Start

1. Add the setup or begin block for the library.
2. Add the action, status, or event blocks needed by your project.
3. For libraries that need polling, the setup blocks automatically add the required loop call when possible.
