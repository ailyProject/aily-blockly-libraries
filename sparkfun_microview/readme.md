## Supported Boards

SparkFun MicroView（仅限该硬件）

## Description

MicroView 是 SparkFun 推出的小型 Arduino + OLED 一体化模块，内置 64x48 OLED 屏，直接插入面包板使用。库提供全局 `uView` 对象访问所有功能。

## Quick Start

1. 将 MicroView 插入面包板
2. 使用「启动 MicroView」块初始化
3. 绘制内容后调用「刷新显示」使画面更新
