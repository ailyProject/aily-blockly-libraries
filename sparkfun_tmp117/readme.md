## Supported Boards

Arduino UNO, Arduino Mega, ESP32

## Description

TMP117 是一款超高精度数字温度传感器，分辨率 0.0078125°C，精度 ±0.1°C（-20°C~50°C），支持连续/单次/关机三种工作模式。

## Quick Start

1. 通过 Qwiic/I2C 连接传感器
2. 使用"初始化 TMP117"积木
3. 等待数据就绪后读取温度
