# SparkFun Qwiic 超声波距离传感器

SparkFun Qwiic 超声波传感器（SEN-17777）的 Blockly 封装库，通过 I2C（默认地址 0x2F）测量距离，单位毫米。

## Library Info

| Field | Value |
|-------|-------|
| Package | @aily-project/lib-sparkfun-qwiic-ultrasonic |
| Version | 0.0.1 |
| Author | SparkFun Electronics |

## Quick Start

1. 通过 Qwiic 连接传感器
2. 初始化后，在循环中读取距离（毫米）
