## Description

WiseChip HUD 是通过 I2C 控制的汽车 HUD 显示模块，支持速度、导航箭头、指南针等多种图标显示。

## Quick Start

1. 通过 I2C 连接 HUD 模块
2. 使用"初始化 WiseChip HUD"积木
3. 使用"图标等级"或"导航"积木控制显示内容
