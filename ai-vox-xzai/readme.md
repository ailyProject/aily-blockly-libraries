# 小智AI语音交互库（ESP32专用）

## 简介
本库是基于 AI Vox 语音交互引擎的 ESP32 专用语音交互库，专为 ESP32 和 ESP32S3 开发板设计，提供便捷的语音识别、交互响应等功能，适用于智能家居、智能设备控制等场景。

## 功能特点
- 支持语音指令识别与解析
- 提供多轮语音交互能力
- 简化的初始化与循环处理接口
- 适配 ESP32/ESP32S3 硬件特性，优化语音处理性能
