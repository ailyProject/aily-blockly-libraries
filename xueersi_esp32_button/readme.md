## Description

OneButton-style Blockly blocks for the Xueersi XiaoMiao ESP32 onboard Up, Down, Left, Right, A and B buttons.

## Quick Start

1. Enable `@aily-project/lib-xueersi-esp32-button` in Aily Blockly.
2. Add the init block in `arduino_setup()`, then use polling or event blocks in `arduino_loop()`.
