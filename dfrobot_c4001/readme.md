# C4001 mmWave Radar

Aily Blockly library for the DFRobot C4001 human presence radar.

## Library Info

| Field | Value |
|-------|-------|
| Package | @aily-project/lib-dfrobot-c4001 |
| Version | 0.1.0 |
| Author | DFRobot |
| Source | https://github.com/DFRobot/DFRobot_C4001 |
| License | MIT |

## Supported Boards

Arduino-compatible boards with I2C or UART.

## Description

Supports presence detection, range and speed data, sensitivity, range, delay, PWM/IO output, and micro-motion settings.

## Quick Start

Initialize C4001 by I2C or UART in setup, choose presence or speed mode, then read detection or target data in loop.
