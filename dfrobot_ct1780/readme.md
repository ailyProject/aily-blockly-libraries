# CT1780 K-Type Temperature

CT1780 is a 1-Wire K-type high-temperature sensor.

## Library Info

| Field | Value |
|-------|-------|
| Package | `@aily-project/lib-dfrobot-ct1780` |
| Version | `0.1.0` |
| Author | DFRobot |
| Source | https://gitee.com/dfrobot/DFRobot_CT1780 |
| License | MIT |

## Supported Boards

Arduino-compatible boards supported by OneWire.

## Description

Searches CT1780 devices on a 1-Wire bus, reads Celsius temperature, and reads the DIP-switch configuration address. OneWire is bundled.

## Quick Start

Initialize the data pin, search for a device, then read temperature or configuration address from the cached device address.
