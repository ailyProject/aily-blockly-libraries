# SparkFun 土壤湿度传感器

I2C 土壤湿度传感器 Blockly 库，适用于植物监测项目。

## Library Info

| Field | Value |
|-------|-------|
| Package | @aily-project/lib-sparkfun-soil-moisture |

| License | MIT |

## Description

通过 I2C 接口读取土壤湿度，支持原始 ADC 值和百分比，内置 LED 指示灯。

## Quick Start

1. 通过 Qwiic/I2C 连接
2. 使用"初始化土壤湿度传感器"积木
3. 读取"百分比"获取 0-100% 湿度值
