## Supported Boards

Arduino UNO, ESP32（3.3V）

## Description

VEML7700 是 Vishay 出品的高精度 I2C 环境光传感器，支持 0~120,000 lux 测量范围，I2C 地址 0x10。

## Quick Start

1. 通过 Qwiic/I2C 连接传感器
2. 使用"初始化 VEML7700"积木
3. 读取 getLux() 获取 lux 值
