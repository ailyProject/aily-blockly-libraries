## Supported Boards

All Arduino-compatible boards supported by the MIDI Library.

## Description

This wrapper uses the default MIDI instance and exposes begin/read, common MIDI send methods, getters, thru mode, and callbacks.

## Quick Start

1. Add the setup or begin block for the library.
2. Add the action, status, or event blocks needed by your project.
3. For libraries that need polling, the setup blocks automatically add the required loop call when possible.
