# SparkFun TeensyView OLED

128×32 像素 SPI OLED 显示屏 Blockly 库，专为 Teensy 系列设计。

## Description

TeensyView 是 SparkFun 为 Teensy 3.x/4.x 设计的 128×32 OLED 附加板，支持文字和图形绘制。

## Quick Start

1. 使用"初始化 TeensyView"积木，指定 5 个 SPI 引脚
2. 调用"打印"积木显示文字
3. 调用"刷新显示"将缓冲区内容输出到屏幕
