# SparkFun UG2856 Transparent OLED

用于 WiseChip UG2856KLBAG01 透明 OLED 的 Blockly 库。

## Library Info

| Field | Value |
|-------|-------|
| Package | @aily-project/lib-sparkfun-hyperdisplay-ug2856klbag01 |

| License | MIT |

## Supported Boards

支持 I2C 或 SPI 的 Arduino/ESP32 开发板。

## Description

该库封装透明 OLED 的 I2C/SPI 初始化、清屏、像素、线条、矩形、圆形、文本、对比度、反相和电源控制。源码已包含 HyperDisplay 与 SSD1309 依赖。

## Quick Start

选择 I2C 或 SPI 初始化块，然后使用单色绘图积木控制显示内容。
