## Description

RFD77402 是 Integrated Device Technology 出品的 I2C 激光测距传感器，最远测量距离约 2m，精度±10%。

## Quick Start

1. 通过 I2C 连接传感器
2. 使用"初始化 RFD77402"积木
3. 调用"执行测量"后读取距离
