# PJON

PJON is a multi-master bus network protocol for local and networked device communication.

## Supported Boards

AVR, ESP8266, ESP32, SAMD, STM32, Teensy, and other PJON-supported Arduino cores.

## Description

This Blockly wrapper focuses on the common SoftwareBitBang transport: create a bus, send text payloads, poll the bus, and react to received packets.

## Quick Start

1. Add the setup or begin block for the library.
2. Add the action, status, or event blocks needed by your project.
3. For libraries that need polling, the setup blocks automatically add the required loop call when possible.
