# 易编离线语音识别（ASR）

适配OpenJumper离线语音识别模块，使用串口接收语音识别结果，支持多种语音指令识别，适用于语音控制、智能家居等场景。

## Library Info
- **Name**: @aily-project/lib-asr
- **Version**: 0.0.1

## Block Definitions

| Block Type | Connection | Parameters (args0 order) | ABS Format | Generated Code |
|------------|------------|--------------------------|------------|----------------|
| `openjumper_asr_init` | Statement | RX_PIN(dropdown), TX_PIN(dropdown) | `openjumper_asr_init(RX_PIN, TX_PIN)` | `` |
| `openjumper_asr_data` | Statement | (none) | `openjumper_asr_data()` | (dynamic code) |
| `openjumper_asr_rincmd` | Value | ASR_CMD(dropdown) | `openjumper_asr_rincmd(0)` | `asr.asrDate ==` |
| `openjumper_asr_state` | Value | (none) | `openjumper_asr_state()` | `asr.WakeUpStatus` |

## ABS Examples

### Basic Usage
```
arduino_setup()
    openjumper_asr_init(RX_PIN, TX_PIN)
    serial_begin(Serial, 9600)

arduino_loop()
    serial_println(Serial, openjumper_asr_rincmd(0))
    time_delay(math_number(1000))
```

## Notes

1. **Initialization**: Place init/setup blocks inside `arduino_setup()`
2. **Parameter Order**: Follows `block.json` args0 order
