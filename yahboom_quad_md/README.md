# Yahboom 四路电机驱动

Yahboom 四路电机驱动模块，支持通过 I2C 或串口（USART）接口控制最多 4 路电机。兼容 520、310、TT 编码、TT 直流及 520-L 型电机。

## 功能概述

- **双模式通信**：支持 I2C 和串口两种通信方式
- **4 路电机控制**：同时控制 M1-M4 四路电机
- **速度/PWM 控制**：带编码器电机可用速度模式，无编码器电机用 PWM 模式
- **编码器数据**：读取 10ms 实时编码器偏移和总编码器计数
- **速度反馈**（串口模式）：获取电机实时速度（mm/s）
- **PID 调节**（串口模式）：设置电机 PID 参数

## 支持的电机类型

| 类型 | 说明 | 推荐参数 |
|------|------|----------|
| 520 电机 | 标准 520 电机 | 减速比 30，脉冲线 11，轮径 67mm，死区 1600 |
| 310 电机 | 310 电机 | 减速比 20，脉冲线 13，轮径 48mm，死区 1300 |
| TT 编码电机 | 带编码器 TT 电机 | 减速比 45，脉冲线 13，轮径 68mm，死区 1250 |
| TT 直流电机 | 无编码器 TT 电机 | 减速比 48，死区 1000（仅 PWM 控制） |
| 520-L 电机 | L 型 520 电机 | 减速比 40，脉冲线 11，轮径 67mm，死区 1600 |

| 设置死区 | 设置电机死区值 |
| 设置脉冲线数 | 设置磁环脉冲线数 |
| 设置减速比 | 设置电机减速比 |
| 设置轮径 | 设置车轮直径（mm） |
| 速度控制 | 控制 4 路电机速度（需编码器电机），范围 -1000~1000 |
| PWM 控制 | 直接控制 4 路电机 PWM 值 |
| 获取 10ms 编码器偏移 | 读取指定电机的 10ms 实时编码器偏移 |
| 获取总编码器计数 | 读取指定电机的总编码器计数 |

| 块 | 说明 |
|---|---|

## 使用流程

### I2C 模式 - 速度控制

1. 拖入"初始化 I2C 电机驱动"，选择地址和电机类型
2. 根据电机类型设置：死区、减速比、脉冲线数、轮径
3. 在循环中使用"速度控制"设置 4 路电机速度
4. 可选：使用"获取编码器"块读取编码器数据

### I2C 模式 - PWM 控制（TT 直流电机）

1. 拖入"初始化 I2C 电机驱动"，电机类型选 TT 直流电机
2. 设置死区和减速比
3. 在循环中使用"PWM 控制"设置 4 路电机 PWM

### 串口模式 - 速度控制 + 数据读取

1. 拖入"初始化串口电机驱动"，选择串口和电机类型
2. 设置电机参数（死区、减速比、脉冲线数、轮径）
3. 使用"数据上报"开启所需的数据类型
4. 在循环中：
   - 使用"收到数据"判断是否有新数据
   - 使用"解析数据"解析收到的数据
   - 读取编码器或速度值

## 注意事项

- **TT 直流电机**（无编码器）只能使用 PWM 控制，不能使用速度控制
- **I2C 模式**默认地址为 0x26，确保接线正确（SDA、SCL）
- **串口模式**使用 115200 波特率，`receive()` 已自动添加到主循环
- 配置参数（死区、脉冲线数等）需放在初始化之后，每条配置间有 100ms 延时
- 支持 Arduino AVR、MegaAVR 和 ESP32 平台
