# Emakefun 电机驱动板

Emakefun Motor Driver Board 是一款基于 PCA9685 PWM 驱动芯片的多功能电机驱动扩展板，通过 I2C 通信，支持 4 路直流电机、8 路舵机和 2 路步进电机控制。

## 功能概述

- **直流电机**：支持 4 路（M1-M4），可调速（0-255）和方向控制
- **舵机**：支持 8 路（S1-S8），角度控制（0-180°），支持速度调节
- **步进电机**：支持 2 路（1-2），支持单步、双步、交错、微步模式
- I2C 通信，默认地址 0x60，支持多板级联

## 块说明

### 初始化

| 块 | 说明 |
|---|---|
| 初始化电机驱动板 | 设置 I2C 地址和 PWM 频率。舵机/直流电机用 50Hz，步进电机用 1600Hz |

### 直流电机

| 块 | 说明 |
|---|---|
| 直流电机运动 | 设置电机速度（0-255）和方向（正转/反转） |
| 直流电机停止 | 刹车（锁定）或释放（自由转动） |

### 舵机

| 块 | 说明 |
|---|---|
| 舵机转到角度 | 设置舵机角度（0-180°） |
| 舵机以速度转到角度 | 以指定速度（1-10）转到目标角度 |
| 舵机当前角度 | 读取舵机当前角度值 |

### 步进电机

| 块 | 说明 |
|---|---|
| 设置速度 | 设置步进电机转速（RPM） |
| 步进运动 | 运动指定步数，可选方向和步进模式 |
| 释放 | 断电释放电机，减少功耗 |

## 使用流程

### 直流电机控制

1. 拖入"初始化电机驱动板"，PWM 频率选 50Hz
2. 拖入"直流电机运动"，选择电机编号、速度和方向
3. 需要停止时使用"直流电机停止"

### 舵机控制

1. 拖入"初始化电机驱动板"，PWM 频率选 50Hz
2. 拖入"舵机转到角度"，选择舵机编号和目标角度
3. 可选使用带速度的版本进行缓慢运动

### 步进电机控制

1. 拖入"初始化电机驱动板"，PWM 频率选 1600Hz
2. 拖入"设置速度"设置 RPM
3. 拖入"步进运动"，设置步数、方向和模式（标准步进电机 200 步 = 1 圈）
4. 运动完成后使用"释放"断电

## 注意事项

- 使用舵机或直流电机时，PWM 频率选择 **50Hz**
- 使用步进电机时，PWM 频率选择 **1600Hz**
- 舵机和步进电机由于 PWM 频率不同，不建议同时使用
- 直流电机可与舵机共用 50Hz 频率
- 步进电机标准 200 步/圈（1.8°/步）
- 支持 Arduino AVR、MegaAVR 和 ESP32 平台
