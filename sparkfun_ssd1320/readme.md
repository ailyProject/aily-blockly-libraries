# SparkFun SSD1320 OLED 显示屏

SPI 接口单色 OLED 显示屏 Blockly 库。

| Field | Value |
|-------|-------|

| Version | 0.0.1 |

| License | MIT |

## Description

SSD1320 驱动的 160×32 像素单色 OLED，通过 SPI 通信。

## Quick Start

1. 连接 RST/DC/CS 引脚（使用默认 SPI 引脚）
2. 使用"初始化 SSD1320"积木
3. 调用"打印"积木在指定坐标显示文字
