# HomeSpan

HomeKit Accessory Protocol implementation for ESP32 on Arduino.

## Supported Boards

ESP32 boards supported by HomeSpan.

## Description

This wrapper builds simple HomeSpan accessory databases and LED-backed lightbulb services.

## Quick Start

1. Add the setup or begin block for the library.
2. Add the action, status, or event blocks needed by your project.
3. For libraries that need polling, the setup blocks automatically add the required loop call when possible.
