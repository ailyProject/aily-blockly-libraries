# M5Cardputer Keyboard

Blockly bindings for the official M5Cardputer keyboard API. Matrix pins and scanning are fixed by the device library.

The change block compares the complete physical key set rather than only the number of pressed keys.

## Description

Blockly bindings for the official M5Cardputer keyboard API. Matrix pins and scanning are fixed by the device library.

## Quick Start

1. Enable `@aily-project/lib-m5stack-cardputer` in Aily Blockly.
2. Add the library blocks, initialize hardware in `arduino_setup()`, then use read/write blocks in `arduino_loop()`.
