# ESP32旋转编码器库

ESP32旋转编码器驱动库适用于esp32，通过I2C通信，实现对旋转编码器信号的高效采集与处理，简化编码器应用开发流程。支持多通道编码器输入、旋转方向与步数精准检测，兼容多种编码器类型。
