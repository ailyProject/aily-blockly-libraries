## Supported Boards

Arduino UNO, Arduino Mega, ESP32

## Description

TMP102 是一款高精度 I2C 数字温度传感器，精度 ±0.5°C，支持低功耗休眠模式和可配置告警输出。

## Quick Start

1. 通过 I2C（SDA/SCL）连接传感器
2. 使用"初始化 TMP102"积木（选择 I2C 地址）
3. 在循环中读取温度值
