# 金逸晨2寸ST7789 TFT屏幕库

适用于学而思ESP32掌机 + 金逸晨2寸ST7789屏幕(240x320)：一键初始化(引脚预设)，绘图/文字/颜色/动画/屏幕信息

## 硬件配置

| 项目 | 配置 |
|------|------|
| 驱动芯片 | ST7789 |
| 分辨率 | 240×320 (横屏320×240) |
| SPI频率 | 40MHz |
| 色序 | BGR |
| 旋转 | 3 (横屏) |
| SPI端口 | HSPI |

### 引脚配置 (预设，不可更改)

| 引脚 | GPIO | 说明 |
|------|------|------|
| MOSI | 23 | SPI数据输出 |
| SCLK | 18 | SPI时钟 |
| CS   | 5   | 片选 |
| DC   | 4   | 数据/命令 |
| RST  | 19  | 复位 |
| MISO | 19  | SPI数据输入(本屏未用) |

## 功能模块

- **初始化**: 一键初始化屏幕
- **绘图**: 画点、画线、画矩形、填充矩形、画圆、填充圆、填充背景
- **文字**: 设置颜色、大小、绘制文字
- **颜色**: 预设16色选择、自定义RGB
- **动画**: 上传GIF/MP4播放(支持阻塞/非阻塞模式)
- **屏幕信息**: 获取宽度、高度

## 使用方法

1. 拖入「TFT屏幕 初始化」块到 setup
2. 使用绘图/文字/动画等块进行操作

## 适配说明

本库基于 lib-tft-screen 修改，主要变更：
- 驱动从 ST7735 改为 ST7789
- 分辨率从 128×160 改为 240×320
- SPI频率从 27MHz 提升到 40MHz
- 色序从 RGB 改为 BGR
- 修复 ST7789_Rotation.h 中 240×320 分辨率在横屏下的偏移问题
