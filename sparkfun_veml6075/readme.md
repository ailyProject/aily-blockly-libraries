## Supported Boards

Arduino UNO, ESP32（3.3V）

## Description

VEML6075 通过测量 UVA 和 UVB 辐照度来计算 UV 指数，I2C 接口（地址 0x10）。

## Quick Start

1. 通过 I2C 连接传感器
2. 使用"初始化 VEML6075"积木
3. 读取 UV 指数或 UVA/UVB 值
