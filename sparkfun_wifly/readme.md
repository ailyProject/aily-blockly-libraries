# SparkFun WiFly WiFi 扩展板

WiFi 网络连接 Blockly 库，支持 TCP 客户端通信。

## Description

WiFly 是基于 RN-131 WiFi 模块的 Arduino 扩展板，通过 SoftwareSerial 通信，支持 TCP/UDP 连接。

## Quick Start

1. 使用"初始化 WiFly 模块"积木（默认 TX=2, RX=3）
2. 用"连接网络"积木输入 SSID 和密码
3. 使用"连接服务器"和"发送"积木进行 TCP 通信
