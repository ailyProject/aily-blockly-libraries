# Adafruit Motor Driver Library

Adafruit motor driver board library supports controlling DC motors and stepper motors, suitable for Arduino control boards

| Field | Value |
|-------|-------|
| Package | @aily-project/lib-afmotor |
| Version | 1.0.0 |

| License | Original license |

## Supported Boards

Arduino AVR, Arduino SAMD, ESP32, RP2040, Arduino UNO R4

## Description

Adafruit motor driver board library supports controlling DC motors and stepper motors, suitable for Arduino control boards

## Quick Start

1. Enable `@aily-project/lib-afmotor` in Aily Blockly.
2. Add the library blocks, initialize hardware in `arduino_setup()`, then use read/write blocks in `arduino_loop()`.
