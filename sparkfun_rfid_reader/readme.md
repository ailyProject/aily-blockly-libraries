# SparkFun UHF RFID 读卡器

同时读取多张 UHF RFID 标签的读卡器 Blockly 库。

## Library Info

| Field | Value |
|-------|-------|
| Package | @aily-project/lib-sparkfun-rfid-reader |
| Version | 0.0.1 |
| Author | SparkFun Electronics |
| Source | https://github.com/sparkfun/SparkFun_Simultaneous_RFID_Tag_Reader_Library |
| License | MIT |

## Description

基于 M6E Nano UHF RFID 模块，支持 ISO18000-6C 协议，可同时读取多张标签。

## Quick Start

1. 连接模块到硬件串口（Serial1 等）
2. 使用"初始化 RFID 读卡器"积木
3. 调用"开始扫描"，循环检查"检测到标签"
