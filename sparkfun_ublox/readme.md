# SparkFun uBlox GPS/GNSS

高精度 I2C GPS/GNSS 模块 Blockly 库。

## Description

支持 u-blox M8、M9、F9 等系列 GNSS 模块，通过 I2C 获取高精度定位数据。

## Quick Start

1. 通过 Qwiic/I2C 连接 GPS 模块
2. 使用"初始化 uBlox GPS"积木
3. 在循环中调用"获取位置数据"后读取经纬度和卫星数
