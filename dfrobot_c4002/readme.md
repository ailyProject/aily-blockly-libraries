# DFRobot C4002 mmWave Radar

Aily Blockly library for the DFRobot C4002 24GHz FMCW radar sensor.

## Supported Boards

UNO, Leonardo/Mega/Micro, UNO R4, XIAO RA4M1, and ESP32 boards with UART.

## Description

C4002 detects motion, static presence, and micro-motion for smart home presence sensing. It supports speed, direction, ambient light, zone gates, adjustable range, calibration, and OUT pin mode.

## Quick Start

Enable the library, place the init block in setup, call update in loop, then read note type and radar values.
