# DFRobot语音识别模块

DFRobot_ASR_M语音识别模块控制库，适用于Arduino、ESP32等开发板。使用I2C接口进行通信，支持循环模式、指令模式和按钮模式三种识别模式，可添加自定义语音词条并识别返回对应的识别号。

## Description

DFRobot_ASR_M语音识别模块控制库，适用于Arduino、ESP32等开发板。使用I2C接口进行通信，支持循环模式、指令模式和按钮模式三种识别模式，可添加自定义语音词条并识别返回对应的识别号。

This library provides 5 Blockly blocks.
