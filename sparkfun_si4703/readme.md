# SparkFun Si4703 FM 收音机

FM 广播接收模块 Blockly 库，支持 76~108 MHz。

## Description

Si4703 是 Silicon Laboratories FM 收音机芯片，支持 RDS，使用 I2C/3-wire 通信。

## Quick Start

1. 连接 RST、SDIO、SCLK 引脚
2. 使用"初始化 Si4703"积木并指定引脚
3. 使用"频道"积木调频（频率×10）
4. 使用"音量"积木设置音量（0-15）
