# BH1750光照传感器

BH1750数字光照强度传感器控制库，适用于Arduino、ESP32等开发板。使用I2C接口读取光照强度数值，将环境光转换为数字信号直接输出勒克斯值，支持连续测量模式和单次测量模式。
