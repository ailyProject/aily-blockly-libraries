# Seeed RPC BLE

Blockly wrapper for Seeed Wio Terminal rpcBLE with BLE server, client, scan, UART, Web Bluetooth battery service, and iBeacon support.

## Supported Boards

Arduino SAMD, Seeeduino:samd

## Description

Blockly wrapper for Seeed Wio Terminal rpcBLE with BLE server, client, scan, UART, Web Bluetooth battery service, and iBeacon support.

## Quick Start

1. Enable `@aily-project/lib-seeed-rpcble` in Aily Blockly.
2. Add the library blocks, initialize hardware in `arduino_setup()`, then use read/write blocks in `arduino_loop()`.
